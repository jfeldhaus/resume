# AI

There are many useful applications of AI technology in software development besides code generation. The screenshot below is from an application that compares the results of a long running stress test executed against two different versions of a database. The graphs compare the progression of processes during the duration of the test.

![](images/ai-compare.jpg)

The text below the graphs is automatically generated by an AI agent (Codex). The agent is provided input, in the form of a JSON document, containing links to all of the output files generated by the test processes. The agent is then instructed by the application to compare the output for each corresponding process and describe any significant differences. In this case the agent has correctly identified that two of the processes failed due to missing shared libraries.
